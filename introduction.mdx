# MyLights Media Browser

MyLights is a self-hosted web application that gives you a fast, modern interface for browsing and playing your local media collection. Originally designed as an Electron desktop app, it was rebuilt on **Next.js** and a **PHP/MySQL** backend to deliver better performance and a richer browser-based experience.

## What is MyLights?

MyLights connects to your local MySQL database of media metadata and presents it through a responsive, YouTube-inspired grid interface. You can browse images, videos, and audio clips; search by title, description, or category; organize content into playlists; and keep track of what you've watched through automatic history tracking.

The system runs entirely on your local machine via **XAMPP**, which bundles Apache (for PHP) and MariaDB (MySQL) into a single install — no cloud accounts or subscriptions required.

## Architecture overview

MyLights uses a classic client-server split optimized for self-hosting:

| Layer | Technology | Role |
|---|---|---|
| Frontend | Next.js 16 + TypeScript + Tailwind CSS | UI, routing, client-side state |
| Backend | PHP 8 (Apache via XAMPP) | REST API for media search and stats |
| Database | MySQL / MariaDB (XAMPP) | Stores media metadata and categories |

```
Browser (Next.js on :4000)
        │
        │ HTTP fetch
        ▼
Apache (XAMPP on :80)
  └── /backend/api/search.php
  └── /backend/api/stats.php
  └── /backend/api/auth.php
        │
        │ mysqli
        ▼
MySQL / MariaDB (XAMPP on :3306)
  └── media_host_db
```
