The MyLights API is a PHP 8 REST API served by Apache via XAMPP. All endpoints accept HTTP GET requests and return JSON.

## Base URL

```
http://localhost/backend/api
```

## General conventions

- **Method:** All endpoints use `GET`
- **Response format:** `Content-Type: application/json; charset=UTF-8`
- **CORS:** `Access-Control-Allow-Origin: *` — open to all origins
- **Authentication:** Not currently required. `auth.php` is a planned feature.

## Error responses

All endpoints return HTTP 500 with a JSON body on failure:

```json
{ "error": "message" }
```

## Endpoints

| Method | Path | Description | Status |
|--------|------|-------------|--------|
| `GET` | [`/search.php`](/api/search) | Search and filter media items | Available |
| `GET` | [`/stats.php`](/api/stats) | Retrieve database-wide media statistics | Available |
| `GET` | [`/auth.php`](/api/auth) | User authentication | Planned |

## TypeScript client

A typed client is available at `frontend/lib/api-client.ts`. It wraps both available endpoints:

```typescript
import { searchMedia, getStats } from '../lib/api-client';

// Search media
const results = await searchMedia('nature', 'Video');

// Fetch stats
const stats = await getStats();
```

<Note>
The client functions return empty arrays or `null` on error rather than throwing, so always check return values before use.
</Note>
